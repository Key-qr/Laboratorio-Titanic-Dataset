# 📊 Laboratorio de Análisis de Datos y Machine Learning

## 📌 Descripción General

Este repositorio documenta el desarrollo de prácticas orientadas al **Análisis Exploratorio de Datos (EDA)**, **Preprocesamiento**, y **Modelado Predictivo**, aplicando técnicas de Ciencia de Datos sobre múltiples datasets reales.

El laboratorio integra procesos estructurados de limpieza, transformación y análisis estadístico, culminando en la implementación de modelos de clasificación supervisada bajo la metodología **SEMMA**, garantizando un enfoque sistemático y reproducible.

## 📂 Datasets Analizados

- 🚢 **Titanic Dataset** – Modelado de supervivencia (clasificación binaria)  
- 🌸 **Iris Dataset** – Clasificación multiclase  
- ❤️ **Heart Failure Dataset** – Predicción de eventos clínicos  
- 📊 **Datos Macroeconómicos** – Análisis descriptivo y tendencias  
- 💎 **Diamonds Dataset** – Análisis estadístico exploratorio  



## 🔬 Metodología Aplicada – SEMMA

El flujo de trabajo se desarrolló siguiendo la metodología:

1. **Sample** – Selección y segmentación de datos relevantes  
2. **Explore** – Análisis estadístico descriptivo y visual  
3. **Modify** – Limpieza, imputación y transformación de variables  
4. **Model** – Entrenamiento de modelos de clasificación supervisada  
5. **Assess** – Evaluación mediante métricas como accuracy, matriz de confusión y validación cruzada  


## 📈 Resultados Técnicos

- ✔️ Tratamiento de valores nulos y outliers  
- ✔️ Codificación de variables categóricas  
- ✔️ Normalización y preparación de datos  
- ✔️ Implementación de modelos supervisados  
- ✔️ Evaluación cuantitativa del rendimiento del modelo  


## 📎 Conclusión

El laboratorio demuestra la aplicación integral del ciclo de vida del análisis de datos, desde la adquisición hasta la evaluación del modelo, aplicando principios de reproducibilidad, claridad estructural y buenas prácticas en ciencia de datos.
